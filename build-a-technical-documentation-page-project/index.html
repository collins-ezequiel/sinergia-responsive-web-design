Introducción

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general.

Es ampliamente utilizado en ciencia de datos, desarrollo web, automatización, y más.

Python es conocido por su sintaxis clara y legibilidad.

print("Hola mundo")
  • Fácil de aprender
  • Comunidad amplia
  • Gran cantidad de librerías
Sintaxis Básica

No se usan llaves ni punto y coma, el bloque de código se define por la indentación.

Los comentarios se hacen con el símbolo #.

# Esto es un comentario if 5 > 3:\n print("Cinco es mayor que tres")
  • No se usan llaves
  • Indentación obligatoria
  • Sintaxis minimalista
Variables y Tipos

No se necesita declarar el tipo de dato, Python lo infiere automáticamente.

Principales tipos: int, float, str, bool, list, dict, tuple.

nombre = "Eze" edad = 25 es_mayor = True
  • Tipado dinámico
  • Mutable vs inmutable
  • Colecciones: list, dict, set
Funciones

Se definen con la palabra clave def.

Las funciones pueden tener parámetros y retornar valores.

def saludar(nombre):\n return "Hola " + nombre print(saludar("Eze"))
  • def
  • return
  • Parámetros y argumentos
Listas y Bucles

Las listas son estructuras que almacenan múltiples valores.

Se pueden recorrer con bucles for o while.

frutas = ["manzana", "banana", "pera"] for fruta in frutas:\n print(fruta)
  • for in
  • while
  • range()
